# experiment-1

## What it is

This project is a polished dark-mode web app concept featuring a marketing landing page, an authentication flow, and a data-rich dashboard, generated with [Potion Kit](https://github.com/uiPotion/potion-kit) - an AI CLI agentic tool for building web projects with [HaroldJS](https://haroldjs.com) and [UIPotion](https://uipotion.com).

## How to run

### 1) Install dependencies

```bash
npm install
```

### 2) Start local development server

```bash
npm start
```

### 3) Build for production

```bash
npm run build
```

The production output is generated in the `build/` directory.
