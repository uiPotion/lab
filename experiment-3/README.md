# experiment-3

## What it is

This project is a landing page with dark mode, blog, chat UI, and login/auth flow, generated with [potion-kit](https://github.com/uiPotion/potion-kit) and the Kimi K2.5 model. It is built with [HaroldJS](https://haroldjs.com) and [UIPotion](https://uipotion.com).

## How to run

### 1) Install dependencies

```bash
npm install
```

### 2) Start local development server

```bash
npm start
```

### 3) Build for production

```bash
npm run build
```

The production output is generated in the `build/` directory.
