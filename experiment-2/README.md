# experiment-2

## What it is

This project is a modern dark/light mode blog experience built with [potion-kit](https://github.com/uiPotion/potion-kit), including post pages, categories, contact form validation, and client-side article search powered by Lunr.js. It is adapted from a UIPotion-inspired playground setup.

## How to run

### 1) Install dependencies

```bash
npm install
```

### 2) Start local development server

```bash
npm start
```

### 3) Build for production

```bash
npm run build
```

The production output is generated in the `build/` directory.
