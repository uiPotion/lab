# uipotion-lab

A lab and playground for web experiments generated based on [UIPotion](https://uipotion.com).

Each subdirectory is a standalone experiment with its own stack and setup.

## Experiments

- **[experiment-1](./experiment-1/)** — Dark-mode landing page, auth flow, and data dashboard. Generate with [potion-kit](https://github.com/uiPotion/potion-kit) and Kimi K2.5 model.

## Prerequisites

Requirements vary by experiment. Typically Node.js and npm; see each experiment’s README for details.

## Usage

Enter any experiment directory and follow its README. Most run with `npm install` and `npm start`.
